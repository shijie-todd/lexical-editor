# 🚀 快速开始 - 表格功能测试

## 第一步：启动项目

```bash
cd D:\code\lexical-editor
npm run dev
```

## 第二步：在浏览器中打开

通常是 `http://localhost:5173`

## 第三步：测试表格功能

### ✅ 方法 1：创建新表格

1. 在编辑器中输入 `/`
2. 输入 `table` 或 `表格`
3. 按回车
4. 表格创建成功！

### ✅ 方法 2：使用已有表格

页面已经预加载了一个测试表格，直接使用即可。

## 第四步：找到操作按钮

**⚠️ 重要：必须先点击单元格！**

```
点击单元格步骤：
1. 用鼠标点击表格中的任意单元格
2. 确保光标在单元格内（会看到闪烁的光标）
3. 这时单元格右上角会出现操作按钮
```

### 按钮位置示意图：

```
┌──────────────────────────┐
│                          │
│  单元格内容        [▼]  │  ← 按钮在这里（右上角）
│                          │
└──────────────────────────┘
```

### 按钮外观：

- 📦 大小：24×24 像素
- 🎨 白色背景
- 🔲 灰色边框
- 🔽 向下箭头图标
- ✨ 有轻微阴影

## 第五步：打开操作菜单

点击 `[▼]` 按钮，会弹出菜单：

```
┌─────────────────────┐
│ ➕ 在上方插入行      │
│ ➕ 在下方插入行      │
├─────────────────────┤
│ ➕ 在左侧插入列      │
│ ➕ 在右侧插入列      │
├─────────────────────┤
│ ➖ 删除行           │
│ ➖ 删除列           │
├─────────────────────┤
│ 🗑️ 删除表格         │
└─────────────────────┘
```

## 🎬 完整演示流程

### 示例：在表格底部添加一行

```
当前表格：
| 姓名 | 年龄 | 职业 |
| 张三 | 28 | 工程师 |
| 李四 | 32 | 设计师 |  ← 点击这一行的任意单元格
```

操作步骤：
1. 点击最后一行（李四那一行）的任意单元格
2. 点击单元格右上角的 `[▼]` 按钮
3. 选择"在下方插入行"
4. 完成！

```
结果：
| 姓名 | 年龄 | 职业 |
| 张三 | 28 | 工程师 |
| 李四 | 32 | 设计师 |
| [空] | [空] | [空] |  ← 新添加的行
```

## 🔍 故障排查

### 问题 1：看不到按钮

**解决方法：**

1. 确保**已经点击了单元格**（光标在单元格内）
2. 刷新页面（Ctrl+F5）
3. 打开浏览器开发者工具（F12）查看是否有错误

### 问题 2：按钮一闪而过

**原因：**点击外部导致菜单关闭

**解决方法：**
1. 先点击单元格
2. 等待按钮出现
3. 慢慢移动鼠标到按钮上
4. 点击按钮

### 问题 3：点击按钮没反应

**检查：**
1. 浏览器控制台是否有错误
2. 是否正确安装了依赖（`npm install`）
3. 开发服务器是否正常运行
